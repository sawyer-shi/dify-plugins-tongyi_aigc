## English Version

### Overview

The TongYi AIGC Plugin ("Plugin") is designed to provide AI-powered image and video generation capabilities using Alibaba Cloud Tongyi's Wanxiang and Qwen models. This privacy policy explains how we handle your data when you use our plugin within the Dify platform.

### License Information

This plugin is licensed under Apache License 2.0, which is a permissive open-source license that allows for the use, modification, and distribution of software. The full license text is available in the LICENSE file included with this plugin.

### Data Collection

**What we DO NOT collect:**
- Personal identification information
- User account details
- Usage analytics or tracking data
- Device information
- Location data
- Cookies or similar tracking technologies

**What we process:**
- Text prompts for image and video generation
- Image files for image-to-image, image-to-video, and image translation operations
- API credentials (API Key) for Alibaba Cloud DashScope services
- Task IDs for video generation status queries

### Data Processing

**External API Processing:**
- All image and video generation requests are sent to Alibaba Cloud DashScope API services
- Text prompts and image files are transmitted to Alibaba Cloud servers for AI processing
- API credentials are used to authenticate requests to Alibaba Cloud services
- Video generation tasks are processed asynchronously on Alibaba Cloud infrastructure

**Local Processing:**
- Plugin configuration and settings are stored locally
- Temporary files may be created during file processing
- No permanent data storage beyond the plugin session

### Data Storage

**No Local Data Retention:**
- We do not store copies of your text prompts after generation
- We do not store copies of your uploaded images after processing
- We do not retain generated images or videos locally
- API credentials are stored securely in Dify's plugin configuration

**Alibaba Cloud Data Handling:**
- Alibaba Cloud processes your prompts and images according to their privacy policy
- Generated content is stored temporarily on Alibaba Cloud servers for retrieval
- Users should review Alibaba Cloud's privacy policy for their data handling practices

### Security Measures

**Data Protection:**
- API credentials are transmitted securely over HTTPS
- All communication with Alibaba Cloud API uses encrypted connections
- Plugin operates within Dify's secure environment
- No unauthorized network connections are made

**Third-Party Services:**
- This plugin relies on Alibaba Cloud DashScope API services
- Users must have valid Alibaba Cloud API credentials
- All AI processing occurs on Alibaba Cloud infrastructure

## 中文版本

### 概述

通义 AIGC 插件（"插件"）旨在使用阿里云通义的万相和通义千问模型提供 AI 图像和视频生成功能。本隐私政策说明了您在 Dify 平台内使用我们的插件时，我们如何处理您的数据。

### 许可证信息

本插件采用 Apache License 2.0 许可证，这是一个宽松的开源许可证，允许使用、修改和分发软件。完整的许可证文本可在本插件附带的 LICENSE 文件中找到。

### 数据收集

**我们不收集的内容：**
- 个人身份信息
- 用户账户详细信息
- 使用分析或跟踪数据
- 设备信息
- 位置数据
- Cookie 或类似跟踪技术

**我们处理的内容：**
- 图像和视频生成的文本提示词
- 用于图生图、图生视频和图像翻译操作的图像文件
- 阿里云 DashScope 服务的 API 凭证（API Key）
- 视频生成状态查询的任务 ID

### 数据处理

**外部 API 处理：**
- 所有图像和视频生成请求都发送到阿里云 DashScope API 服务
- 文本提示词和图像文件传输到阿里云服务器进行 AI 处理
- API 凭证用于验证对阿里云服务的请求
- 视频生成任务在阿里云基础设施上异步处理

**本地处理：**
- 插件配置和设置存储在本地
- 文件处理期间可能会创建临时文件
- 除插件会话外，不进行永久性数据存储

### 数据存储

**不保留本地数据：**
- 生成后我们不存储文本提示词的副本
- 处理后我们不存储上传图像的副本
- 我们不在本地保留生成的图像或视频
- API 凭证安全存储在 Dify 的插件配置中

**阿里云数据处理：**
- 阿里云根据其隐私政策处理您的提示词和图像
- 生成的内容临时存储在阿里云服务器上以供检索
- 用户应查看阿里云的隐私政策了解其数据处理做法

### 安全措施

**数据保护：**
- API 凭证通过 HTTPS 安全传输
- 与阿里云 API 的所有通信使用加密连接
- 插件在 Dify 的安全环境中运行
- 不进行未经授权的网络连接

**第三方服务：**
- 本插件依赖阿里云 DashScope API 服务
- 用户必须拥有有效的阿里云 API 凭证
- 所有 AI 处理都在阿里云基础设施上进行
